# GenAI Tutorials

## Overview

This repository contains a collection of tutorials and examples for Generative AI models. The tutorials are designed to be easy to follow and provide a practical introduction to the field of Generative AI.

## Structure

The repository is organized into the following directories:

- **AutoGen**: Tutorials related to the AutoGen library.
- **Basics**: Basic examples for Generative AI, including:
    - **LLM-Basics**: Introductory examples for using Large Language Models (LLMs).
    - **RAG-Basic**: Basic examples for Retrieval-Augmented Generation (RAG).
- **Google_ADK**: Tutorials related to Google's ADK.
- **LangGraph**: Tutorials related to the LangGraph library.
- **Project**: Example projects using Generative AI.

## Installation

To run the examples, you will need to install the required dependencies. For example, to install the dependencies for the `LLM-Basics` project, navigate to the `Basics/LLM-Basics` directory and run:

```bash
pip install -r requirement.txt
```
